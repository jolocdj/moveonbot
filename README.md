# 💔 MoveOnBot

MoveOnBot is a Discord bot designed to help you move on, resist relapse, and process emotions — without actually messaging the person you shouldn’t.

It gives smart, context-aware responses based on what you’re feeling, so it doesn’t feel repetitive or robotic.

---

## Features

- 💬 **Context-aware replies** (not just random)
- 🧠 Detects emotions like:
  - missing someone
  - relapse urges
  - sadness
  - loneliness
  - anger
- 📝 Helps process **unsent messages**
- 🚫 Prevents impulsive texting
- 📈 Tracks **no-contact progress**

---

## 🤖 Commands

| Command | Description |
|--------|-------------|
| `/moveon` | Get a reminder to move forward |
| `/relapse` | Stop yourself from sending that message |
| `/unsaidthoughts` | Share what you wanted to say (but didn’t) |
| `/nocontact` | Track your no-contact days + feelings |
| `/replycheck` | Check if your message is worth sending |
